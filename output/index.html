Installation ¶

The preferred way to install the Yii framework is through composer.

Either run

composer global require "fxp/composer-asset-plugin:^1.2.0"
composer require yiisoft/yii2

or add

"yiisoft/yii2": "~2.0.0",

to the require section of your composer.json.
